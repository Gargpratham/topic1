Topic 1 Practical Activity

Summary of Activity 1:

In activity 1, we had to create a mind map on IoT application and answer some related questions. The questions are listed below:

  • What is an IoT application?
  • Can you think of some examples?
  • What are the main features needed in an IoT application?

Summary of Activity 2:

In activity 2, we had to create a mind map considering a scenerio of a smart building and answer some related questions. The questions are listed below:

  • What does smart mean?
  • Think of some smart building scenarios where this will be beneficial.
  • What are the parts needed for it to work?

Summary of Activity 3:

In activity 3, we had to create a mind map control interface required for a smart building IoT application and answer some related questions. The questions are listed below:

  • What are the high level features needed for this interface?
  • Think about who will use the interface. Who are the users?
  • What are the parts needed to make this work?

List of IoT applications:

An IoT application is a prebuilt software service that is used to connect different iot devices together on internet. It provides users an interface to control, record and store data of the devices.

Below is a list of some of the IoT applications:

  • Google Smart home
  • Xiaomi Remote Control Application
  • Arduino IoT remote
  • Fitness tracking applications
  • IP security cameras
